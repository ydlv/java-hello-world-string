# Java hello world string

This repository contains a class that defines a single public static final string that says "Hello, World!". That's it.

## Why

To test publishing to JitPack.

## How to use

`HelloWorldString.HELLO_WORLD`

## Dependencies

...Are you really asking?

OK, there's one - JUnit, and only in the test scope.
